# 💕 محادثتنا — WhatsApp Chat Gift

تطبيق ويب رومانسي لعرض محادثة واتساب كهدية مميزة.

## المميزات
- 🌹 تصميم رومانسي مع نجوم وقلوب متحركة
- 💬 عرض المحادثة بفقاعات ملونة
- 🔍 بحث في المحادثة
- 📊 إحصائيات شاملة (رسائل، كلمات، إيموجي)
- 📅 خط زمني تفاعلي
- 📌 نقطة توقف (وقفنا هنا)
- 🌐 واجهة عربية RTL بالكامل

## الاستخدام
1. صدّر محادثتك من واتساب كملف `.txt`
2. ضع الملف في المجلد الرئيسي باسم `chat.txt`
3. افتح `index.html` في المتصفح (أو استخدم سيرفر محلي)

## النشر
ارفع المجلد على [Netlify Drop](https://app.netlify.com/drop) للحصول على رابط مباشر.
